## 知乎爬虫系统
#### 主要功能
- [x] 采集单个问题的答案信息
- [ ] 采集单个问题的基本信息
- [ ] 采集单个问题的相关问题
- [ ] 采集单个问题的回答者信息
#### 支持的数据导出模式
- [x] mysql
- [x] mongo
- [x] xls
#### 支持的连接模式
- [x] 使用代理
- [ ] 不使用代理
#### 配置导出方式
```angular2html
在setting.py中，将需要用到的导出方式设置为1，其余设置为0
# ---------- EXPORT INFO ---------- #
USE_MYSQL = 1
USE_MONGO = 0
USE_XLS = 0
# ---------- EXPORT INFO ---------- #
```
